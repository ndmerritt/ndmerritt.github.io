Introduction to HTML

HTML stands for HyperText Markup Language.

It is the standard language used to create web pages.

<!DOCTYPE html> <html></html>
  • HTML uses tags
  • Tags define structure
HTML Document Structure

An HTML document has a defined structure.

The structure includes head and body sections.

<head></head> <body></body>
  • Head contains metadata
  • Body contains visible content
Common HTML Elements

There are many commonly used HTML elements.

Paragraphs, headings, and links are essential elements.

<p>Paragraph</p>
  • <h1> to <h6>
  • <a> links
Working with Forms

Forms allow user input in web pages.

They are essential for login and registration systems.

<form></form>
  • Input fields
  • Submit buttons
Semantic HTML

Semantic HTML improves readability and accessibility.

It clearly describes the meaning of content.

<section></section>
  • <article>
  • <footer>
