Project Overview

My application is an improved professional portfolio for my mom that showcases her design projects from a variety of clients she has had. It will mainly serve as an updated and more appealing version of the portfolio she had made years ago, with it having new features like hover actions and embedded media. The intended users of the site will be employers looking to hire, and the updated design will hopefully improve her image to these employers.

Page Design

Home Page

This is the main landing that users will first see when navigating to the site. It will have cards with thumbnails of some featured projects that when hovered over, will display a brief description of the project. The header for the page will have a dropdown when hovering over portfolio that will display the pages for each client. The audience for this page is users, and there won't be any data being entered by users on this page.

Bassett

This page will feature all projects related to her work at Bassett furniture. Each one will have an image along with a detailed description of the purpose it fulfilled. The audience for this page is users, and there won't be any data being entered by users on this page.

CloudBurst

This page will feature the project related to her work for the client CloudBurst. This specific project will feature embedded video on work she made for the client's website. The audience for this page is users, and there won't be any data being entered by users on this page.

AIP

This page will feature the project related to her work for the client AIP. Similar to CloudBurst, this page might also have an embedded video for the project. The audience for this page is users, and there won't be any data being entered by users on this page.

CollegeAdmit

This page will feature the project related to her work for the client CollegeAdmit. The audience for this page is users, and there won't be any data being entered by users on this page.

About Me/Contact

This page will contain some information about Deanna and her career path to becoming a graphic designer. The page will also display her email and have a button that takes you to another page for sending her a message. The audience for this page is users, and there won't be any data being entered by users on this page.

Send a Message

This page will allow users to directly send messages to her via the website. It will have fields for a name, sender email, and message, all which have validation. The audience for this page is users.

Dynamic Functionality

Some examples of dynamic functionality with JavaScript on the website include:

  • The thumbnail gallery on the main page that will show descriptions for each project when they are hovered over (example: Urban Outfitters)
  • The dropdown menu in the nav bar for Portfolio, revealing the different clients (example: Target)
  • The form area in the Send a Message page taking input data from the user and storing it (example: Suvidha)
