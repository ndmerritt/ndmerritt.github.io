Markdown Key

  • # text → Heading 1 (<h1>)
  • ## text → Heading 2 (<h2>)
  • ### text → Heading 3 (<h3>)
  • > text → Blockquote
  • ![alt](url) → Image
  • [text](url) → Link
  • **text** or __text__ → Bold
  • *text* or _text_ → Italic
